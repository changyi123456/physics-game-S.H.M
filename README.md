# Physics Game S.H.M

A browser RPG prototype that wraps periodic motion simulations into a mission-based mystery game.

## Tech Stack

- React
- TypeScript
- Vite
- Phaser

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This project is configured for GitHub Pages under:

```text
https://changyi123456.github.io/physics-game-S.H.M/
```
